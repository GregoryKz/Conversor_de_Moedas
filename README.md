Conversor de Moedas • Offline

Descrição:
Um conversor de moedas moderno e responsivo, que permite calcular rapidamente valores entre diversas moedas sem depender de APIs externas. Possui histórico de conversões, troca rápida de moedas e interface clean com chips de moedas populares.

Sobre o projeto

Este projeto é um Conversor de Moedas desenvolvido com HTML, CSS e JavaScript puro. Ele funciona 100% offline, usando taxas pré-definidas para garantir conversões consistentes e confiáveis.

Funcionalidades:

Conversão entre moedas populares (USD, BRL, EUR, GBP, JPY, ARS, CAD, AUD, CNY)

Botão de troca rápida de moedas (↔︎)

Chips de moedas populares para conversão instantânea

Histórico das últimas conversões

Botão para copiar o resultado

Botão para limpar o formulário

Design moderno com gradientes e efeito glass

Responsivo para desktop e mobile

Tecnologias utilizadas

HTML5 – Estrutura da página

CSS3 – Estilo moderno com gradientes, flexbox e grid

JavaScript – Lógica do conversor e manipulação do DOM

Font Awesome – Ícones para botões e interface

Estrutura do projeto
conversor-moedas/
│
├─ index.html        # Estrutura da página
├─ style.css         # Estilo do projeto
└─ script.js         # Lógica da conversão offline

Como usar

Abra index.html no navegador.

Insira o valor que deseja converter.

Escolha as moedas de origem e destino.

Clique em Converter.

Use os botões ↔︎, Limpar ou Copiar conforme necessário.
